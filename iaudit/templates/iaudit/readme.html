Instructions for Windows:

  1. Download WindowsScan.bat
  2. Double click to launch the script. Give it a few minutes to complete.
  3. Wait for the command to execute successfully. Look for the file name starting with AUDI____.csv on your desktop
  4. Upload the scan results here

Instructions for Mac:

  1. Use the keyboard shortcut Command + Space to open the Spotlight Search Box.
  2. Type Terminal in the search box to find the terminal app.
  3. Double click or hit enter when you see the Terminal app in the search results.
  4. Once the terminal window appears, copy and paste the following command in the terminal window:
    ls -h /Applications/ > ~/Desktop/AUDI-MAC$(system_profiler SPHardwareDataType | awk '/Serial/ {print $4}').csv
  5. Wait for the command to execute successfully. Look for the file name starting with AUDI-MAC____.csv on your desktop
  6. Upload the scan results here
